# kapua

A fully decentralised storage, compute, database and networking platform, in C++11.

## Building

### Dependencies:

Please see [dependencies](docs/dependencies.md) in the docs.

### Build

```sh
git clone https://github.com/tomdionysus/kapua
cd kapua
mkdir build
cd build
cmake ..
make
```
